Pokemon Log Details Web API

self-made project
Github Repo Live Site!

Pokemon Log Details API was a project made by optimizing the vast amount of data from PokeAPI. Technologies used in this project are HTML, CSS, and JavaScript. To make the project run smoothly, it used the best practice of Asynchronous JavaScript. I also used my experience playing Pokemon, giving retro taste to design and the best way to show PokeAPI data.

The features:

  • Auto-generate the first twenty Pokemon.
  • When you click the Pokemon box, it will show the pokemon attribute details.
  • You can click the abilities and moves to give more explanation about it.
  • You can search pokemon name by using index or name of the Pokemon

Pig Game

following along project
Github Repo Live Site!

Pig Game is the second mini-project of Jonas Schmedtmann's JavaScript Course. It applies simple DOM implementation and JavaScript logic. The game is about getting as many scores from the dice. Your points are the accumulation of numbers in your turn. If you get the number 1, your points will be gone, and it's the next turn for another player. The key to this game is you don't become greedy and be patient. Tech used in this project is HTML, CSS, and JavaScript.
